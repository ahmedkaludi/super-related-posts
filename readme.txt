== Description ==

Add a highly configurable list of related posts to any posts. Related posts can be based on any combination of word usage in the content, title, or tags.

### Features 

* Lot of options available to customize the display & layout of displayed posts
* Matching of current post's category, tags, content and author
* Exclude posts by author, ID, category, tags and custom fields
* Output related posts in posts, widgets and RSS
